Have a question? Check out the FAQ

We’ve compiled the most common questions to help you understand how our platform works quickly and easily.

What are the roles available for users on this platform?

There are three roles: Project Manager, Team Member, and Special Team Member. Each role has different access levels and responsibilities.

What’s the difference between public and private projects?

In public projects, all members can view complete project details. In private projects, users can only see the tasks assigned to them and the projects they were added to.

Can I switch a project from public to private later?

Yes, project visibility can be changed by the Project Manager at any time through the project settings.

What security measures protect login and authentication?

We use secure HTTPS protocols, password hashing, session management to safeguard user access.

Who can see all details in private projects?

Only Project Managers have full access to all private project details. Team members and special members see limited, relevant information only.

What’s the role of a Special Team Member?

Special Team Members are usually added for expert tasks. They get access only to the parts of the project they’re involved in.
